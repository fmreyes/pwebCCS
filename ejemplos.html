Que es PHP ?

1. Descripción Hipertextos Preprocessor, los servidores utilizan este es un lenguaje de scripting, siendo uno de los más utilizados.

2. Características (5) Se puede manejar una gran extensión de documentación, por la cantidad de usuarios. Variedad de herramientas para aprender. Módulos externos para mejorar la aplicación web. Se puede separar la estructura. Flexibilidad y seguridad. 3. Ventajas (5) Compatibilidad multiplataforma. Características de programación orientada a objetos. Integración con HTML, CSS, JavaScrip, etc. Desde el ámbito laboral muchas ofertas de trabajo para quien desarrolla este lenguaje. Existen varios Frameworks PHP como Laravel, Symmofoly, Codelgniter, etc. 4. Desventajas (5) No es tan seguro debido a su código abierto. No es adecuado para aplicaciones gigantes basadas en contenido. El uso de más funciones del marco y las herramientas PHP provoca en las aplicaciones web sea más deficiente. Es muy difícil de administrar porque no es modular competente, imita las características del lenguaje Java. 5. Tipo de licencia Gratuita. 6. Análisis de curva de aprendizaje Fácil aprendizaje y uso. 7. Ejemplo de uso Se utilizan en plataformas como Wikipedia, WordPress, Facebook entre otras 8. Página Oficial https://www.php.net/

Que es LARAVEL ?

1. Descripción. Es un Fremework PHP, es un marco web de código abierto en una arquitectura MVC (Model View Controller)

2. Características (5) Tiene una herramienta de línea de comandos. (Artisan) Bibliotecas modulares y orientadas a objetos, previamente instalados. Eloquent ORM (Mapeo relacional de objetos) Templete Engine. Socialite para simplificar la autenticación OAuth con pFacebbol, GitHub, Google, Etc. 3. Ventajas (5) Desarrollo activo Comunidad activa Buena documentación Motor de plantillas Blade: sofisticado y sensillo 4. Desventajas (5) La instalación se hace mediante Composer. Solo soporta a partir de PHP 5.4 5. Tipo de licencia Licencia MIT, Software libre permisiva. 6. Análisis de curva de aprendizaje Desde su lanzamiento en 2012 la tendencia es al alza. Pero es una curva de aprendizaje más lenta Codelgniter 7. Ejemplo de uso. Koel aplicación para escuchar música. Vuedo es un proyecto para la creación de blogs. Laracom es un proyecto que crea tiendas online. 8. Página oficial. https://laravel.com/
